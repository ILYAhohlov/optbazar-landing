# ОптБазар.рф - Landing Page

Красивый и современный landing page для маркетплейса оптовых овощей и фруктов с рынка Азия-Сибирь.

## Структура проекта

- `index.html` - основная HTML страница
- `style.css` - стили CSS
- `script.js` - JavaScript функциональность
- `images/` - папка для изображений
- `README.md` - этот файл

## Особенности

✅ Адаптивный дизайн для всех устройств
✅ Современный минималистичный стиль
✅ Структура по принципу AIDA
✅ Формы для партнёров и подписки
✅ Плавные анимации и переходы
✅ SEO-оптимизированная разметка

## Блоки страницы

1. **Hero** - главный экран с призывом к действию
2. **Проблема + Выгоды** - сравнение проблем и решений
3. **Как работает** - пошаговая инструкция
4. **Для партнёров** - форма подключения продавцов
5. **Call to Action** - финальный призыв к действию

## Запуск

Просто откройте `index.html` в браузере.

## Изображения

Добавьте следующие изображения в папку `images/`:
- `market-hero.jpg` - фото рынка Азия-Сибирь для главного блока
